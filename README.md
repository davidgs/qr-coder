## Install

Download the latest release and drag the application to your `/Applications` folder (macOS).

## Known Issues

The QR Code dynamically updates as you change  text and settings. **However**, when changing the shape of the 'eyes' it does *not* update the QR Code. You will have to either change the text or another setting to update the QR Code.

If you have a fix for this, please feel free to submit a PR.
